Az 1-esek és 0-ák világa

A kettes számrendszer, más néven bináris számrendszer, az informatika és a digitális elektronika alapját képezi. Ez a számrendszer kizárólag két számjegyet használ: a 0-t és az 1-et. A kettes számrendszer különösen fontos a számítógépek működésében, mivel a digitális eszközök alapvető működési elve a két állapot (bekapcsolt és kikapcsolt) megkülönböztetésén alapul.

Kettes számrendszerből egy táblázat segítségével tudunk átalakítani tízesbe:

1 1 0 1 0 0 0 1
1 2 4 8 16 32 64 128
1 2 0 8 0 0 0 128 139

a felső és az alsó számot össze kell szorozni.
